# Submission Reflection

## What I Learned

### Technical Skills
- **NestJS Framework**: Gained experience with modular architecture, dependency injection, and decorators
- **TypeORM**: Learned ORM concepts, entity relationships, and database operations
- **TypeScript**: Improved understanding of types, interfaces, and type safety
- **File Upload Handling**: Implemented multer integration for receipt and avatar uploads
- **REST API Design**: Created CRUD endpoints following REST conventions

### Development Practices
- **Test-Driven Development**: Started implementing unit and integration tests
- **Code Organization**: Structured code into modules and followed naming conventions
- **Error Handling**: Basic error handling and validation
- **Documentation**: Created comprehensive specifications and API documentation

## Challenges Faced

### Database Configuration
- Setting up MySQL connection with TypeORM
- Understanding entity relationships and migrations
- Handling database errors and connection issues

### File Upload Implementation
- Configuring multer with disk storage
- Handling multipart form data in NestJS
- Managing file paths and serving uploaded files

### Frontend-Backend Integration
- Connecting HTML forms to API endpoints
- Handling asynchronous requests with JavaScript
- Updating UI dynamically after API calls
